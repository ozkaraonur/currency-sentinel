# Currency Sentinel

## Overview
A lightweight Python application that interacts with the **ExchangeRate-API** to fetch and display real-time currency conversion rates. This project demonstrates API integration, secure data handling, and input sanitization.

## Features
- **Live Data Retrieval**: Fetches real-time exchange rates using RESTful API calls.
- **Dynamic Queries**: Users can specify a base currency or view all available global rates.

## Technical Skills Applied
- **Python**: Modular code structure and input validation.
- **API Security**: Prevention of credential leaking via placeholders.
- **JSON Parsing**: Processing hierarchical data structures.

## Installation & Usage
1. Obtain an API Key from [exchangerate-api.com].
2. Clone the repository: `git clone https://github.com/ozkaraonur/currency-sentinel.git`
3. Install dependencies: `pip install requests`
4. Set your key in `main.py` and run: `python main.py`
